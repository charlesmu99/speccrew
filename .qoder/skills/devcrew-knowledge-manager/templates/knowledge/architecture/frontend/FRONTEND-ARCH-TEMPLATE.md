## 目录结构

```
[前端根目录]/src/
├── [接口层目录]/    # API 请求封装，按业务模块分文件
├── [静态资源目录]/  # 静态资源（图片、样式变量）
├── [公共组件目录]/  # 公共组件
├── [页面目录]/    # 页面级组件
├── [状态管理目录]/  # 全局状态管理
├── [路由目录]/    # 路由配置
├── [组合逻辑目录]/  # 可复用的组合式逻辑
└── [工具目录]/    # 工具函数
```
